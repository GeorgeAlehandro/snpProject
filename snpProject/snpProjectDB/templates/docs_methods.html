Methods

From where is the data extracted?

The data is extracted from public scientific articles that study the GWAS associations of the different traits and SNPs as part of population studies.

How can I identify strongly related traits/SNPs?

By looking at P-value and choosing the lowest ones for strong associations. ALso checking for the association of this specific SNP to the same trait by checking the SNP page can give the user an idea about the confidence for the repetition of the association of this SNP to the trait.

How was this webpage built?

This webpage was built using Django and the queries you see are based on SQLite databases that use the power of Python datatables coupled with Django in order to make the search interactive and server-side.
